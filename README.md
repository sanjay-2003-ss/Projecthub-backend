# ProjectHub Backend

Backend API for the ProjectHub platform. This service handles user authentication, project management, database operations, and all server-side logic used by the ProjectHub Frontend.

## Features

- User authentication and authorization (JWT)
- CRUD API for projects, tasks, and categories
- Secure route protection
- Error-handling middleware
- MongoDB database with Mongoose models
- Clean and modular folder structure

## Tech Stack

- **Node.js**
- **Express.js**
- **MongoDB + Mongoose**
- **JWT Authentication**
- **dotenv for environment handling**
- **Nodemon for development**
